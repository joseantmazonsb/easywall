easywall for iptables

easywall is a software for easy operation of firewalls.
The software was developed in the modern programming language python3.
We are constantly working on improving the software.
Currently easywall is still in the beta version.
If you like the software, we would be glad about financial support.
If you have found a bug, you can report it to us in a GitHub issue or in Discord.
